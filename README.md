# 🎮 Gamer Death District

Jogo de plataforma 2D desenvolvido com Godot Engine 4.6.

![Godot](https://img.shields.io/badge/Godot-4.6-478CBF?logo=godot-engine&logoColor=white)
![GDScript](https://img.shields.io/badge/GDScript-355570?logo=godot-engine&logoColor=white)
![Platformer](https://img.shields.io/badge/2D-Platformer-success)

## ✨ Funcionalidades

- Movimentação e combate
- Sistema de vida
- Dano flutuante
- Coleta de score
- Inimigos com IA
- Pulo duplo
- Fase única/modo sobrevivência
